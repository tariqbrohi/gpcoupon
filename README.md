1. Install packages first by running `yarn` or `npm install`

2. Migrate and generate by running `yarn prisma db push` and `yarn prisma generate` respentively.

Signin to [Mongodb Atlas](https://account.mongodb.com/account/login?_ga=2.160330750.2138064780.1661466344-1984879242.1650475693) and create a free version of Mongodb Cluster.
